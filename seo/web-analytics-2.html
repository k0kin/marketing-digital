Frequently Asked Questions

What is web analytics?
Web analytics is the measurement, collection, analysis and reporting of data about website usage in order to understand and optimize web usage.
How can I use web analytics for digital marketing?
Web analytics can be used to measure the performance of your digital marketing campaigns, such as page views, time spent on site, click-through rates, conversions, etc., in order to better understand what works best and how to improve future campaigns.
What are the benefits of using web analytics?
The benefits of using web analytics include gaining insights into website user behavior that can inform decisions and strategies related to digital marketing campaigns; improving website design; tracking progress towards goals; and increasing ROI from ad spend.
What tools are available for measuring web analytics?
Popular tools for measuring web analytics include Google Analytics, Adobe Analytics (formerly Omniture/SiteCatalyst), Yahoo! Web Analytics (formerly IndexTools), Coremetrics and IBM Digital Analytics (formerly Coremetrics).
How often should I analyze my website's performance with web analytics?
It is recommended that you regularly review your website's performance with web analytics at least once a month so that you can identify trends or changes over time.
